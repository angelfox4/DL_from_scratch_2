## [밑바닥부터 시작하는 딥러닝 2 스터디]
 
- 목표 : 한 달동안 책 1권을 끝내기
- 목적 : 딥러닝 개념에서 나아가 파이썬 코드로 구현을 통해 깊게 이해합니다.
- 내용 : numpy를 이용하여 NLP, RNN에 대해 공부하고 구현해 보는 실습을 진행합니다.

## 요구사항
소스 코드를 실행하려면 아래의 소프트웨어가 설치되어 있어야 합니다.

* 파이썬 3.x
* NumPy
* Matplotlib

※ Python은 3 버전을 이용합니다.

## 실행 방법

각 장의 디렉터리로 이동한 후 파이썬 명령을 실행하세요(**다른 디렉터리에서는 제대로 실행되지 않을 수 있습니다!**).

```
$ cd ch01
$ python man.py
```
